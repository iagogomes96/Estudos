Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> [Não semântica]

para termos em destaque devemos usar a tag <strong> [Semântica]

Itálico/Ênfase

Nesa frase, temos um termo em itálico usando a tag <i> (não semântica)

para termos com ênfase devemos usar a tag <em> (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto deletado

Esse é um texto deletado usando a tag <del> para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Esse é um texto inserido usando a tag <ins> para dar uma ênfase e indicar que ele foi adicionado depois.

Também existe a forma não semântica que é utilizando a tag <u>

Subscrito e sobrescrito

Podemos criar um texto com subscrito usando a tag <sub> e um texto com sobrescrito usando a tag <sup>

Exemplo subscrito: x20+3

Exemplo sobrescrito: H2O
